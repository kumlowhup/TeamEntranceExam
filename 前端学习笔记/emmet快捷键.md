## EMMET语法和快捷键

| 快捷键    | 内容                     |
| --------- | ------------------------ |
| ctrl n    | 新建文件                 |
| ctrl s    | 保存文件                 |
| ！tab     | 生成html基本骨架         |
| ctrl +或- | 调整视图大小             |
| 标签名*n  | 生成n个标签              |
| >         | 生成父子级关系标签       |
| $         | 顺序自增                 |
| +         | 兄弟关系级标签           |
| `ti`      | `text-indent`            |
| w100      | `width: 100px;`          |
| lh20      | `line-height: 26px;`     |
|           |                          |
|           |                          |
| tdn       | `text-decoration: none;` |
|           |                          |
|           |                          |
|           |                          |
|           |                          |
|           |                          |
