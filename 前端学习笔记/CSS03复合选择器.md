## css07**复合选择器**

**一、后代选择器**<br/>

```CSS
ol li {
    color: #fff;
}
```

父级元素 子级元素空格隔开

**<u>二、子选择器（重要）</u>** 

​	`div>p { 样式声明 }` 选择div里面所有最近一级p标签元素

**<u>三、并集选择器(重要)</u>**

​	`u1,div{ 样式声明 }` 同时选择u1和div

四、伪类选择器

链接伪类选择器用冒号隔开

`a:link` 选择未被访问的链接<br/>`a:hover` 鼠标指针位于其上<br/>`a:visited`已被访问的链接<br/>`a:active`鼠标按住不放的链接
